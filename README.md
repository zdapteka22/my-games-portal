## Как сайт появится в поиске Google / Яндекс

1. Открой сайт: https://zdapteka22.github.io/my-games-portal/
2. Google: https://search.google.com/search-console — добавь сайт и sitemap:
   `https://zdapteka22.github.io/my-games-portal/sitemap.xml`
3. Яндекс: https://webmaster.yandex.ru — добавь сайт и ту же sitemap.
4. Запроси индексацию главной страницы.
5. Поиск «портал игр» — очень конкурентный запрос. Быстрее сайт находится по:
   - `портал игр zdapteka`
   - `портал игр github`
   - `site:zdapteka22.github.io`

Появление в выдаче обычно занимает от нескольких дней до нескольких недель.
