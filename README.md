# Портал игр (локальный сайт)

Готовый сайт **без GitHub**. Всё лежит у тебя на компьютере.

## Как открыть (любой браузер)

1. Дважды нажми файл **`ОТКРЫТЬ_ПОРТАЛ.bat`** в этой папке  
   или открой в браузере: **http://127.0.0.1:8765/**
2. Не закрывай чёрное окно сервера, пока играешь.

## Что внутри
